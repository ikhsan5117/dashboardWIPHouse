# Stats Icons Directory

This directory contains PNG icons for the dashboard stats widgets.

## Required Files:

1. **expired.png** - Icon for "Already Expired" items
   - Recommended size: 60x60px or 120x120px (will be scaled automatically)
   - Should represent expired/expired items (e.g., warning sign, clock, calendar)

2. **near-expired.png** - Icon for "Near Expired" items
   - Recommended size: 60x60px or 120x120px
   - Should represent items expiring soon (e.g., warning triangle, clock)

3. **shortage.png** - Icon for "Shortage" items
   - Recommended size: 60x60px or 120x120px
   - Should represent low stock/shortage (e.g., empty box, low battery, exclamation)

4. **over-stock.png** - Icon for "Over Stock" items
   - Recommended size: 60x60px or 120x120px
   - Should represent excess stock (e.g., upward arrow, full box, plus sign)

5. **normal.png** - Icon for "Normal" items
   - Recommended size: 60x60px or 120x120px
   - Should represent normal/good status (e.g., checkmark, green circle, thumbs up)

## Icon Guidelines:

- **Format**: PNG with transparent background
- **Style**: Flat design or outline style works best
- **Colors**: Icons will be displayed on colored backgrounds, so use white or light colors
- **Consistency**: All icons should have similar style and weight
- **Size**: Icons will be automatically scaled, but 60x60px is optimal

## Download Sources:

- **Flaticon**: https://www.flaticon.com/
- **Icons8**: https://icons8.com/
- **Feather Icons**: https://feathericons.com/
- **Heroicons**: https://heroicons.com/

## Implementation:

The icons are automatically applied via CSS classes:
- `.expired-icon` → expired.png
- `.near-expired-icon` → near-expired.png
- `.shortage-icon` → shortage.png
- `.over-stock-icon` → over-stock.png
- `.normal-icon` → normal.png

## Responsive Sizing:

- Desktop: 60x60px
- Tablet: 50x50px
- Mobile: 45x45px
- Small Mobile: 40x40px
